# CZ Agent

一个基于公开权威资料的研判工具，自动追踪公开活动动向、识别新提法变化、孵化研究切口。

## 功能模块

1. 动向雷达 —— 公开活动与关注点变化追踪
2. 公开信号流 —— 近期公开信息汇总
3. 切口孵化 —— 基于信号自动生成候选研究切口
4. 信号雷达 —— 高频提法与主题热度分析

## 本地运行

直接用浏览器打开 `index.html`，或启动本地服务：

```bash
python3 -m http.server 8765
```

然后访问 `http://127.0.0.1:8765/`。

## 数据更新

每日定时抓取公开权威资料，自动完成「抓取 → 分析 → 切口重算 → 发布」流水线，所有数据均保留原始链接、发布日期与抓取时间。
