# 民盟市委参政议政赋能平台

一期 MVP，用于验证“公开资料自动研判 -> 关注点变化 -> 切口孵化 -> 成果转化”的产品形态。

## 当前版本

- 不接入民盟历史研究平台。
- 不设置盟员线索投稿入口。
- 只围绕公开权威资料做信号识别、专题研判和切口孵化。
- 支持桌面网页和手机端响应式访问。

## 功能模块

1. 领导关注点与政策变化雷达
2. 公开信号流
3. 自动切口孵化
4. 成果转化工作台
5. 公开权威信源清单

## 本地运行

可以直接用浏览器打开 `index.html`。为了让链接、滚动和后续数据接口更接近正式环境，也可以启动本地服务：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/
```
